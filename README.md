# 30 Day JS Challenge

Roadmap latihan 30 hari, 1-2 jam/malam. Tiap folder = 1 hari = 1 project kecil.

## Cara pakai struktur ini
Tiap hari, copy folder `day-01-calculator` (ganti nama sesuai project), lalu isi `index.html`, `style.css`, `script.js`.
Struktur HTML/CSS/JS sengaja dipisah 3 file biar kebiasaan terbentuk dengan baik dari awal.

## Roadmap

1. Calculator sederhana
2. To-do list (tambah, hapus, tandai selesai)
3. Digital clock + alarm sederhana
4. Quote generator (array of quotes)
5. Color palette generator (random hex)
6. Quiz app (pilihan ganda, skor di akhir)
7. Stopwatch / timer

### Minggu 2 — DOM Lanjutan & LocalStorage

9. Expense tracker sederhana

12. Image slider/carousel
13. Accordion FAQ
14. Tab switcher

### Minggu 3 — Async & API
15. Random joke generator
16. Weather app
17. GitHub user finder
18. Currency converter
19. Movie search app
20. Infinite scroll / load more
21. Recipe finder

### Minggu 4 — Project Kompleks & OOP
22. Memory card game (OOP)
23. Tic-tac-toe (OOP)
24. Drum kit / sound board
25. Form validator (regex)
26. Drag and drop kanban
27. Pomodoro timer

30. Personal dashboard (gabungan beberapa fitur)

## API Gratis (untuk Minggu 3, tanpa API key)
- Joke: https://official-joke-api.appspot.com/random_joke
- Weather: https://open-meteo.com/ (gratis, tanpa key)
- GitHub user: https://api.github.com/users/USERNAME
- Currency: https://www.exchangerate-api.com/ (perlu daftar gratis untuk key) atau https://open.er-api.com/v6/latest/USD (tanpa key)
- Movie: https://www.omdbapi.com/ (perlu API key gratis)
- Recipe: https://www.themealdb.com/api.php (gratis, tanpa key)

## Tips Konsistensi
- Functionality dulu, styling secukupnya. Jangan kebanyakan polish CSS di hari yang sama.
- Kalau 1-2 jam belum kelar, lanjut besok — jangan paksa selesai malam itu.
- Push ke GitHub tiap hari, 1 commit 1 hari. Bagus untuk portfolio + motivasi visual progress.
- Hari dengan fetch API: siapkan API key di awal sebelum mulai ngoding biar gak buang waktu.
